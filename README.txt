//RABBYT (VIRTUAL PET GAME)

RABBYT is a simple, console-based virtual pet simulator built with C# and .NET.

This project was created as my first C# application to explore object-oriented programming concepts, class design, and basic game logic. It highlights my ability to apply object-oriented programming principles in C#, designing and maintaining stateful systems, writing readable and maintainable code, and building a complete, playable application from scratch.

//PROJECT OVERVIEW

The goal of RABBYT is to keep your virtual rabbit alive for as many years as possible by balancing their hunger and happiness levels.

The game demonstrates:
> Class encapsulation and property management.
> State changes over time using the aging system.
> Interactive user input and feedback.
> ASCII art for simple but dynamic UI presentation.

//FEATURES

> Pet Management - feed and play to maintain your rabbit's wellbeing.
> Aging System - Each turn represents a passing year.
> Dynamic ASCII Art - Displays the rabbit's mood visually.
> Game Over Conditions - based on hunger or happiness reaching critical levels.
> Interactive Console Menu - simple input-driven gameplay.

//TECHNOLOGIES USED
> Language - C#
> Framework - .NET Console Application
> Concepts - object-oriented programming, game loops, encapsulation
